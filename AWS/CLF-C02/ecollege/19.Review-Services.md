# In-Scope AWS Services and Features
Below is a brief explanation of each AWS service and feature covered in the AWS Cloud Practitioner exam. 

## Analytics
1. **Amazon Athena**<br>A serverless query service that allows users to run SQL queries directly on data stored in Amazon S3 without managing infrastructure.
2. **AWS Data Exchange**<br>A data marketplace that allows business to find, subscribe to, and use third-party datasets from various providers.
3. **Amazon Elastic MapReduce(EMR)**<br>A big data processing service that enables users to run Apache Spark, Hadoop and other frameworks for large-scale data processing.
4. **AWS Glue**<br>A serverless ETL (Extract, Transfer, Load) service that helps clean, prepare and move data from analytics and machine learning.
5. **Amazon Kinesis**<br>A service for real-time data streaming and analytics, commonly used for log monitoring, IoT data processing and video streaming.
6. **Amazon Managed Streaming for Apache Kafka (Amazon MSK)**<br>A fully managed Apache Kafka service that enables business to stream real-time data for applications.
7. **Amazon OpenSearch Service**<br>A managed search and analytics service based on OpenSearch and Elasticsearch used for log analytics, full-text search and security monitoring.
8. **Amazon QuickSight**<br>A cloud-powered business intelligence (BI) service for visualising data through interactive dashboards
9. **Amazon Redshift**<br>A fully managed data warehouse that enables business to run complex quieries on large datasets.

<br>

## Application Integration
10. **Amazon EventBridge**<br>A Serverless event bus that connects AWS services and triggers workflows in response to events.
11. **Amazon Simple Notification Service (SNS)**<br>A messageing service taht allows users to send push notification, SMS and email alerts.
12. **Amazon Simple Queue Service (SQS)**<br>A message queuing service that enables asynchronous communication between microservices.
13. **AWS Step Functions**<br>A workflow orchestration service that automates the execution of AWS services.

<br>

## Business Applications
14. **Amazon Connect**<br>A cloud-based contact centre solution that allows businesses to set up customer support call centres.
15. **Amazon Simple Email Service (SES)**<br>A scalable email-sending service used for marketing, transactional emails, and automated notifications.

<br>

## Cloud Financial Management
16. **AWS Billing Calculator**<br>A cost allocation tool that allow AWS Organizations to create custom billing reports.
17. **AWS Budgets**<br>A tool that enables business to set spending limits and receive alerts when they approach their budget.
18. **AWS Cost and Usage Report (CUR)**<br>A deatiled billing report that tracks AWS service usage and associated costs.
19. **AWS Cost Explorer**<br>A cost analysis tool that allows users to visualise and forecast AWS spending trends.
20. **AWS Marketplace**<br>An online store where business can find and purchase third-party software solutions.

<br>

## Compute
21. **AWS Batch**<br>A batch processing service that runs large-scale computing workloads efficiently.
22. **Amazon EC2**<br>A virtual machine service that allows businesses to run applications on AWS-managed instances.
23. **AWS Elastic Beanstalk**<br>A PaaS that automates the deployment of web applications.
24. **Amazon Lightsail**<br>A simplified cloud computing service for small business and developers.
25. **AWS Local Zones**<br>Extends AWS services to specific geographic locations to reduce latency.
26. **AWS Outpost**<br>A hybrid cloud solution that extends AWS infrastructure on-premises.
27. **AWS Wavelength**<br>Brings AWS compute closer to mobile network for low-latency applications.

<br>

## Containers
28. **Amazon Elastic Container Registry (ECR)**<br>A private container image repository to store and manage Docker images.
29. **Amazon Elastic Container Service (ECS)**<br>A fully managed container orchestration service for running Docker Applications.
30. **Amazon Elastic Kubernetes Service (EKS)**<br>A managed Kubernetes service for deploying and managing containerised applications.

<br>

## Database
35. **Amazon Aurora**<br>A high-performance relational database that supports MySQL and PostgreSQL.
36. **Amazon DynamoDB**<br>A NoSQL dabase for fast, highly scalable applications.
37. **Amazon MemoryDB for Redis**<br>A fully managed Redis-compatible in-memory database.
38. **Amazon Neptune**<br>A graph database service used for social networks, fraud detection, and knowledge graphs.
39. **Amazon Relational Database Service (RDS)**<br>A fully managed database that supports MySQL, PostgreSQL, SQL Server and more.

## Develoepr Tools
AWS provides tools for building, testing and deploying applications, including:
  - **AWS AppConfig** - Manages application configurations
  - **AWS CLI** - Automates AWS services via commands
  - **AWS Cloud9** - Cloud-based integrated development environment (IDE)
  - **AWS CloudShell** - A browser-based CLI
  - **AWS CodeArtifact** - Manages software dependencies
  - **AWS CodeBuild** - Compiles source code and runs tests
  - **AWS CodeCommit** - A Git-based repository service
  - **AWS CodeDeploy** - Automates application deployments
  - **AWS CodePipeline** - Continuous Integration and Deployment (CI/CD) tool
  - **AWS CodeStar** - Simplifies software development workflows
  - **AWS X-Ray** - Traces application performance issues

<br>

## End User Computing
AWS provides virtual desktop and application streaming services:
  - **Amazon AppStream 2.0** - Streams desktop applications to users
  - **Amazon WorkSpaces** - Cloud-based virtual desktops
  - **Amazon WorkSpaces Web** - Provides secure web-based access

<br>

## Frontend Web and Mobile
AWS provides tools for developing and deploying web/mobile applications:
  - **AWS Amplify** - Backend and hosting for web and mobile apps
  - **AWS AppSync** - Managed GraphQL service to real-time apps
  - **AWS Device Farm** - A testing service for mobile and web apps

<br>

## Internet of Things (IoT)
  - **AWS IoT Core** - Connects and manages IoT devices
  - **AWS IoT Greengrass** - Runs AWS services on edge devices

<br>

## Machine Learning
AWS offers AI and ML services such as:
  - **Amazon SageMaker** - Builds and trains ML models
  - **Amazon Rekognition** - Image/video analysis
  - **Amazon Polly** - Convert text to speech
  - **Amazon Translate** - Language translation
  - **Amazon Comprehend** - Natural language processing (NLP)
  - **Amazon Lex** - AI-powered chatbots
  - **Amazon Textract** - Extract text from documents

<br>

## Serverless
AWS offers serverless computing:
  - **AWS Lambda** - Runs code without provisioning servers
  - **AWS Fargate** - Runs containers without managing servers

<br>

## Storage
AWS provides scalable storage solutions:
  - **Amazon S3** - Object storage
  - **Amazon EBS** - Block storage
  - **Amazon EFS** - File storage
  - **Amazon FSx** Windows and Lustre file systems
  - **AWS Backup** - Automated backups
  - **AWS Storage Gateway** - Hybrid cloud storage
