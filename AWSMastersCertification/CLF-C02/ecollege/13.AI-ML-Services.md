# Identify AWS Artificial Intellegence (AI) and Machine Learning (ML) Services and Analytics Services

### Introduction to AWS AI/ML and Analytics Services
AWS provides a range of Artificial Intelligence (AI), Machine Learning (ML), and Data Analytics services that help business automate processes, extract insights from data and build intelligent applications.

##### Key topics covered:
- **AI/ML Services** - Amazon SageMaker, Amazon Lex, Amazon Kendra and others
- **Analytics Services** - Amazon Athena, Amazon Kinesis, AWS Glue, Amazon QuickSight
- **Coosing the right AI/ML or Analytics service for different workloads**


AWS AI/ML and Analytics services help business with tasks such as:
- Predictive modelling
- Data processing and transformation
- Speech and text analysis
- Real-time analytics and data visualisation

<br>

## AWS AI/ML Services

AWS provides fully managed AI/ML services that allow developers and business to build, train and deploy ML models without requiring deep expertise

### Amazon SageMaker
Amazon SageMaker is a fully managed machine learning service that allows developers to quickly build, train and deploy ML models

**Best for:**
- **Developers and data scientists** who need to train and deploy ML models
- **Automating ML workflows** without managing infrastructure
- **Experimenting and fine-tuning ML models** with build-in tools

<br>

### Amazon Lex (Conversational AI)
Amazon Lex is a service for building chatbots and voice assistants powered by the same AI as Alexa

**Best for:**
- **Chatbots for customer support**
- **Automated voice interfaces** for applications
- **Integrating AI-powered conversations** into applications

<br>

### Amazon Kendra (Intelligent Search)
Amazon Kendra is an AI-powered search engine that improves search accuracy and relevance across enterprise documents

**Best for:**
- **Corporate knowledge bases** and document search
- **Customer support and helpdesk automation**
- **Improving internal seach within organisations**

<br>
