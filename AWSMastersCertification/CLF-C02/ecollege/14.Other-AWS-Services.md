# Identify Services from Other AWS Service Categories

### Introduction 
 AWS offers various services across categories beyond core computing, storage, networking and databases. These services help business with application integration, business applications, customer engagement, developer tools, end-user computing, frontend and mobile development and IoT.


**This section covers:**
- **Applciation Integration Services** - Amazon EventBridge, SNS, SQS
- **Business Application Services** - Amazon Connect, Amazon SES
- **Customer Engagement Services** - AWS Activate, AWS IQ, AWS Managed Services, AWS Support
- **Developer Tools** - AWS CodePipeline, AWS CodeDeploy, AWS Cloud9, AWS X-Ray, and more
- **End-User Computing Services** - Amazon WorkSpaces, AppSteam 2.0
- **Frontend and Mobile Services** - AWS Amplify, AWS AppSync
- **IoT Services** - AWS IoT Core, AWS IoT Greengrass

<br>

### AWS Application Integration Services
Application integration services help AWS and external applications communicate and exchange data asynchronously. These services ensure reliable, scalable, and decoupled communication

| **Service**                            | **Description**                                                                          | **Best For**                                        |
|----------------------------------------|------------------------------------------------------------------------------------------|-----------------------------------------------------|
| **Amazon EventBridge**                 | Event-driven service that routes events between AWS services and SaaS applications       | Automating workflows and real-time event processing |
| **Amazon Simple Notification Service** | Fully managed publish/subscribe messaging service for sending notifications and messages | Sending SMS, email and mobile push notifications    |
| **Amazon Simple Queue Service**        | Message queue service for decoupling components on distributed applications              | Managing synchronous workloads and queuing messages |

**Chosing the Right Service:**
- **Need to send push notifications to users?** -> Use SNS
- **Need to queue messages between application components?** -> Use SQS
- **Need event-driven automation?** -> Use EventBridge

<br>

### AWS Business Application Services
AWS provides business application services facilitationg communications, customer support, and email automation

| **Service**                           | **Description**                                                                                | **Best For**                                                   |
|---------------------------------------|------------------------------------------------------------------------------------------------|----------------------------------------------------------------|
| **Amazon Connect**                    | Cloud-based contact centre solution for customer service                                       | Managing inbound/outbound customer interactions                |
| **Amazon Simple Email Service (SES)** | Scalable email-sending service for transactional emails, marketing campaingns and notification | Sending bulk emails, marketing campaigns, and automated emails |

**Chosing the Right Service:**
- **Need a cloud-based call centre for customer service?** -> Use Amazon Connect
- **Need an email-sending service for marketing or notifications?** -> Use Amazon SES

<br>


### AWS Customer Enablement Services
AWS provides services to support startups, maange cloud operations and offer expert consulting

| **Service**                    | **Description** | **Best For** |
|--------------------------------|-----------------------------------------|--------------|
| **AWS Activate for Startups**  | Provides startups with AWS credits, training and support | New business using AWS |
| **AWS IQ**                     | Connects customers with AWS-certified experts for consulting | Hiring AWS professional for architecture and development |
| **AWS Managed Services (AMS)** | AWS experts manage cloud operations (monitoring, patching, security) | Enterprises needing fully managed AWS environemnts |
| **AWS Support**                | Provides technical support plans (Basic, Developer, Business, Enterprise) | Getting AWS help based on support tiers |
