### Billing Support and Pricing Information
#### AWS Billing Support and Information
AWS provides **billing and pricing support** through the AWS Billing Dashboard, AWS Support, and pricing calculators

| **Billing Resource**                | **Purpose**                                                                           |
|-------------------------------------|---------------------------------------------------------------------------------------|
| **AWS Billing Dashboard**           | Provides detailed billing reports, invoices and usage summaries                       |
| **AWS Support for Billing**         | AWS provides billing and account support through different AWS Support Plans          |
| **AWS Cost and Usage Report (CUR)** | Generate detailed billing reports that show AWS usage and costs at the resource level |
| **AWS Pricing Calculator**          | It helps estimate AWS service costs before deployment                                 |

<br>

#### Pricing Information for AWS Services
AWS follows a pay-as-you-go pricing model, where users pay only for the resources they consume

| **Pricing Model**           | **Description**                                                   | **Example Use Case**                                       |
|-----------------------------|-------------------------------------------------------------------|------------------------------------------------------------|
| **On-Demand Pricing**       | Pay for compute, storage, and database usage per hour or second   | Short-term workloads, testing applications                 |
| **Saving Plan**             | Commit to consistent usage for lower costs over 1 or 3 years term | Long-running applications with predictable workloads       |
| **Spot Instances**          | Purchase space compute capacity at a discount (up to 90%)         | Batch processing, fault-tolerant workloads                 |
| **Reserved Instances (RI)** | Commit to a specific instance typefor up to 75% discount          | Steady-state workloads (e.g., databses, long-running apps) |
| **Free Tier**               | Offer limited free usage of AWS services                          | Testing AWS services with no upfront cost                  |

### AWS Organizations and Consolidated Billing
#### AWS Organizations
AWS Organizations is a service that allows busienss to manage multiple AWS accounts under a single entity


#### AWS Consolidated Billing
AWS Consolidated Billing lets organisations combine multiple AWS accounts into a single billing account, which provides:
- **One Billing for All Accounts** - All accounts in an organisation share a single monthly bill.
- **Cost Savings via Volume Discounts - AWS applies discounts** based on the total usage across all linked accounts
- **Separate Costs Allocation** - Costs can be divided per account, department, or team

<br>

#### Use Case for AWS Organizations
- **Companies managing multiple AWS accounts** (e.g. dev, testing, and prod accounts)
- **Business that want centralised billing** while maintaining spearate resource ownership
- enterprise that need cost visibility accross multiple teams
