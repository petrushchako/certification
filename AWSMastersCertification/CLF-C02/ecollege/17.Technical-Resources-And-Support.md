# Identify AWS technical Resources and AWS Support options

### AWS Technical Resources and Documentation
AWS provides various technical resources to help customers understand, implement, and manage cloud services. These resources are available on official AWS websiteand are cathegorised into different formats, including whitepapers, blogs and knowledge centres.

#### AWS Official Documentation
- **AWS Documentation** : https://docs.aws.amazon.com/
- **AWS Whitepapers** Guides on cloud best practices, security, architecture and operational excellence : https://aws.amazon.com/whitepapers/ 
- **AWS Blogs** Technical articles and updates from AWS experts : https://aws.amazon.com/blogs/ 
- **AWS Knowledge Center** Common questions and troubleshooting guides : https://aws.amazon.com/premiumsupport/knowledge-center/
- **AWS re:Post** Community-driven Q&A platform for AWS-related topics : https://repost.aws/
