# Identify AWS technical Resources and AWS Support options

### AWS Technical Resources and Documentation
AWS provides various technical resources to help customers understand, implement, and manage cloud services. These resources are available on official AWS websiteand are cathegorised into different formats, including whitepapers, blogs and knowledge centres.

#### AWS Official Documentation
- **AWS Documentation** : https://docs.aws.amazon.com/
- **AWS Whitepapers** Guides on cloud best practices, security, architecture and operational excellence : https://aws.amazon.com/whitepapers/ 
- **AWS Blogs** Technical articles and updates from AWS experts : https://aws.amazon.com/blogs/ 
- **AWS Knowledge Center** Common questions and troubleshooting guides : https://aws.amazon.com/premiumsupport/knowledge-center/
- **AWS re:Post** Community-driven Q&A platform for AWS-related topics : https://repost.aws/

### AWS Support Plans
AWS provides different plans tailored for customer needs, from bacis troubleshooting to account management

#### AWS Support Plan Tiers
1. **Basic Support (Free)**
   - AWS documentation, whitepapers, and forums
   - AWS re:Post for communicaty support
   - Service health dashboard access
2. **Developer Support**
   - Basic plan benefits + access to AWS support engineers
   - Email-based technical support during business hours
   - AWS Trusted Advisor (limited checks)
3. **Business Support**
   - 24/7 phone, email and chat support from AWS engineers
   - AWS Trusted Advisor (full checks and recommendations)
   - AWS Health Dashboard is used to monitor AWS services
4. **Enterprise On-Ramp Support**
   - Personalized guidance from AWS support
   - Access to an assigned AWS Technical Account Manager (TAM)
   - Proactive monitoring and cost optimization advice
5. **Enterprise Support**
   - Highest level of support with a dedicated TAM
   - Mission-critical support with a 15 minute response time for urgent issues
   - Infrastructure event management and architectural reviews

### AWS Support Center and Trusted Advisor
#### AWS Support Center
- Centralized hub for managing AWS cases
- Direct access to AWS customers service and technical support
- Integrated with AWS Health Dashboard for tracking service status

<br>

#### AWS Trusted Advisor
- Automated tool providing best practice recommendations on:
  - **Cost Optimization**: Identifying underutilised resources
  - **Performance**: Improving efficiency
  - **Security**: Enhancing cloud security
  - **Fault Tolerance**: Ensuring high availability
  - **Service Limits**: Monitoring usage to prevent disruptions

<br><br><br>

### AWS Health Dashboard and AWS Health API
#### AWS Health Dashboard
- Display real-time information about AWS service status and outages
- Provides insights into how service disruptions impact user environments
- Accessible via AWS Console or API

<br>

#### AWS Health API
- Enables programmatic access to AWS Health Dashboard data
- Helps automate monitoring and issue resolution

<br><br><br>

### AWS Trust and Safety Team
AWS has a dedicated Trust and Safety team responsible for:
- Investigating and addressing abusive or fraudulent use of AWS resources
- Helping customers comply with regulatory requirements
- Reporting misuse of AWS via: https://aws.amazon.com/decurity/report-abuse/

### AWS Partner Network (APN)
The AWS Partner Network (APN) consists of third-party vendors who provide software solutions, consulting and integration services to AWS customers

#### AWS Partner Types
- **Independent Software Vendors (ISVs)**: Companies that offer software solutions through AWS Marketplace
- **System Integrators (SIs)**: Firms that help customers deploy and manage AWS solutions

#### Benefits of AWS Partnership
- **Training & Certification**: Exclusive training and AWS certifications
- **Partner Events**: Networking opportunities and early access to AWS innovations
- **Discounts**: Volume discounts and marketing support

#### AWS Marketplace
AWS MArketplace provides access to third-party software solutions for:
  - **Cost Management**: Tools for tracking AWS expenses
  - **Governance & Compliance**: Security and compliance solutions
  - **Entitlement Management**: Licensing and access control tools

### AWS Professional Services and Solutions Architects
#### AWS Professional Services
AWS proides professional consulting to help enterprises design, deploy and optimise AWS solutions. Services include:
  - Migration planning
  - Security and compliance guidance
  - Performance optimisation

#### AWS Solutions Architects
AWS Solutions Architects provide technical guidance and best practices to help organizations
  - Design scalable architectures
  - Implement security and compliance controls
  - Optimize costs and performance
