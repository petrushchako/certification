# AWS Cloud Practitioner Exam - Key Technologies and Concepts

### 1. **APIs (Application Programming Interface)**
APIs allow applications to communicate programatically with AWS services. AWS provides APIs compute, storage, networking and security, enabling automation and integration

<br>

### 2. Benefit of Migrating to AWS Cloud
Migrating to AWS offers benefits like:
- **Scalability** - Easy to scale resources up or down
- **Cost Savings** - Pay only for what you use
- **Security** - Built-in security and compliance tools
- **Reliability** - High availability and fault tolerance

<br>

### 3. AWS Cloud Adoption Framework (AWS CAF)
AWS CAF provides a structured approach to cloud adoption with six key perspectives
  1. **Business** - Aligns cloud strategy with business goals
  2. **People** - Focuses on skill development
  3. **Governance** - Defines policies and compliance
  4. **Platform** - Addresses cloud architecture
  5. **Security** - Ensures cloud security
  6. **Operations** - Optimizes operational processes

<br>

### 4. AWS Compliance
AWS helps customers mees global compliance standard such as:
  - GDRP (General Data Protection Regulation)
  - HIPAA (Health Insurance Portability and Accountability Act)
  - ISO27001 (Information Security Standard) AWS provides compliance reports and audit tools via AWS Artifact

<br>

### 5. Compute
AWS Compute services provide scalable virtual servers and serverless computing:
  - Amazon EC2 - Virtual MAchines in the cloud
  - AWS Lambda - Serverless computing for event-driven applications
  - AWS Fargate - Serverless containers

<br>

### 6. Cost Management
AWS offers tools to monitor and optimise cloud spending:
  - AWS Cost Explorer - Analyze AWS usage and costs
  - AWS Budgets - Sets cost limits and alerts
  - AWS Pricing Calculator - Estimates AWS costs

<br>

### 7. Databases
AWS offers managed database services
  - Amazon RDS - Relational databases (MySQL, PostgreSQL, SQL Server)
  - Amazon DynamoDB - NoSQL key-value store
  - Amazon Redshift - Data warehouseing for analytics

<br>

### 8. Amazon EC2 Instance Types
AWS EC2 instances available in different pricing models:
  - On-Demand - Pay-as-you-go pricing
  - Reserved Instance (RI) - Discounted pricing for long-term usage
  - Spot Instances - Uses spare capacity at a lower cost

<br>

### 9. AWS Global Infrastructure
AWS operates data centres worldwide, organised into:
  - AWS Regions - Separate geoppolitical areas (e.g., us-east-1, eu-west-1)
  - Availability Zones (AZs) - Isolated data centres within a region
  - Edge Locations - Content delivery points for Amazon CloudFront

<br>

### 10. Infrastructure as Code (IaC)
IaC allows automating cloud infrastructure deployment using:
  - AWS CloudFormation - Declarative templates for infrastructure
  - Terraform - Open-source infrastructure management tool
