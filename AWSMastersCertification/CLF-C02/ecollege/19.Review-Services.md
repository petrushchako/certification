# In-Scope AWS Services and Features
Below is a brief explanation of each AWS service and feature covered in the AWS Cloud Practitioner exam. 

## Analytics
1. **Amazon Athena**<br>A serverless query service that allows users to run SQL queries directly on data stored in Amazon S3 without managing infrastructure.
2. **AWS Data Exchange**<br>A data marketplace that allows business to find, subscribe to, and use third-party datasets from various providers.
3. **Amazon Elastic MapReduce(EMR)**<br>A big data processing service that enables users to run Apache Spark, Hadoop and other frameworks for large-scale data processing.
4. **AWS Glue**<br>A serverless ETL (Extract, Transfer, Load) service that helps clean, prepare and move data from analytics and machine learning.
5. **Amazon Kinesis**<br>A service for real-time data streaming and analytics, commonly used for log monitoring, IoT data processing and video streaming.
6. **Amazon Managed Streaming for Apache Kafka (Amazon MSK)**<br>A fully managed Apache Kafka service that enables business to stream real-time data for applications.
7. **Amazon OpenSearch Service**<br>A managed search and analytics service based on OpenSearch and Elasticsearch used for log analytics, full-text search and security monitoring.
8. **Amazon QuickSight**<br>A cloud-powered business intelligence (BI) service for visualising data through interactive dashboards
9. **Amazon Redshift**<br>A fully managed data warehouse that enables business to run complex quieries on large datasets.
