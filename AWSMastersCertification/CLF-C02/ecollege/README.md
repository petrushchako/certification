# AWS Certified Practitioner Exam Preparation

### AWS CLI configuration
- Install AWS CLI:
  ```sh
  brew install awscli
  ```
- Configure credentials:
  ```sh
  aws configure

  #######################################
  # Provide the following details:      #
  #######################################
  # AWS Access Key ID        [*********]
  # Default region name:     [eu-west-1]
  # AWS Secret Access Key    [*********]
  # Default output format    [json]
  ```

> Credentials will be store in `~/.aws/credentials`
