# Install and Configure Kubernetes

### Lab Overview
In this lab, you’ll practice the installation and basic configuration of a Kubernetes cluster. When you’re finished, you’ll have set up a Kubernetes cluster. Furthermore you have the configuration ready to deploy your first test application running on this cluster.

### Learning Objectives
1. Receive Kubernetes Init Information
2. Join a Worker Node to Your Cluster
3. Assign Role to Worker Node and Create Namespace
4. Run an Application on your Kubernetes Cluster
5. Scale the Nginx Deployment to Two Replicas
